## Compilation, 20 points

* No errors or warning

Your functions compiles without generating any errors or warnings.

* Some warnings

Your functions compile but the compiler generates some warnings.

* Errors

You didn't submit anything or your functions do not compile.

## Run, 30 points

* Correct

Your implementation passes all unit tests.

* Close

Your implementation fails some unit tests.

* Incorrect

Your implementation fails most unit tests.

## Memory leaks, 30 points

* No leaks

Your implementation does not leak any memory.

* Some leaks

Your implementation leaks some memory.

* A sieve

No calls to free().

## Check for out-of-memory condition, 10 points

* All calls checked

You check the return value of all calls to malloc().

* Some calls checked

You check the return value of some calls to malloc().

* No checks

You do not bother check any return values.

## Initialization, 10 points

* All memory initialized

All fields in all objects are properly initialized before use.

* Some memory initialized

Some fields in some objects are properly initialized before use.

* No initialization

No object is initialized.
